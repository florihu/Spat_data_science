Validate imputation

util.imp_valid(data, n_folds=5, print_output=False)

Default 5-fold cross validation. For a given dataset three different kinds of imputations are compared 1. simple 2. k uniform 3. k distance

Parameters:

  • data – this is again the raw data

  • n_folds – number of folds. The number of times the dataset is splited and testing is executed.

Returns:

DataFrame with the scores for the alternatives. Evaluated scores =[R2, NMRSE, PBIAS]
